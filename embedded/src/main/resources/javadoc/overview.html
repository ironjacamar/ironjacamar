IronJacamar Embedded

The IronJacamar Embedded configuration allows developers to use a Java EE Connector Architecture container inside their application, or use it for a unit testing infrastructure.

IronJacamar Embedded provides a full Java EE Connector Architecture standalone environment, as defined by Section 3.5 in the specification.

Furthermore it provides integration to the following technologies:

  • Arquillian
  • ShrinkWrap
  • Byteman

See the user guide for furher information.

Package overview

The IronJacamar Embedded configuration contains different packages, which focuses on different parts of the setup.

org.jboss.jca.embedded

This is the main package of the embedded platform with core classes.

This package should be considered advanced usage.

org.jboss.jca.embedded.arquillian

This package contains the classes to control configuration and integration of IronJacamar Embedded with Arquillian.

org.jboss.jca.embedded.dsl

This package contains generic classes for DSL integration.

org.jboss.jca.embedded.dsl.datasources10.api
